Simon Game

This website hosts a classic Simon Game. Players need to remember and repeat the sequence of lights and sounds. The game gets progressively more challenging as the sequence gets longer.

Getting Started

To run this project, you first need to Clone this project.

To Run Locally

  1. Open your terminal and navigate to the project directory.
  2. Run a local server (e.g., using live-server or simply open index.html in your browser).

Project Structure

The project consists of the following files and folders:

  • index.html - The main HTML file that contains the structure of the game.
  • style.css - The CSS file that contains the styling for the game.
  • game.js - The JavaScript file that contains the logic for the game.
  • sounds/ - A folder that contains the sound files used in the game.

To Run on Server

  1. Set up and configure your preferred hosting service (e.g., GitHub Pages, Netlify, Vercel).
  2. Upload the project files (index.html, style.css, game.js, and the sounds folder) to the hosting service.
  3. Ensure the game works correctly on the deployed URL.
